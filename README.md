# 🛒 E-Commerce Application using Spring Boot

A full-stack E-Commerce web application built using Spring Boot, Thymeleaf, Spring Data JPA, and MySQL. The application provides a complete online shopping experience with user authentication, product management, shopping cart functionality, seller management, and an admin dashboard.

---

## 🚀 Features

### User Features

* User Registration and Login
* Browse Products by Category
* Add Products to Cart
* View Cart and Checkout
* Contact Support

### Seller Features

* Seller Dashboard
* Add New Products
* Edit Existing Products
* Manage Product Listings

### Admin Features

* Admin Dashboard
* Manage Users
* Manage Sellers
* Manage Products
* Create Seller Accounts

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Navanathpattekar/Navanathpattekar-E-Commerce-Application-using-Spring-Boot.git
```

### Navigate to Project

```bash
cd Navanathpattekar-E-Commerce-Application-using-Spring-Boot
```

### Configure Database

Update `application.properties`:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/ecommerce_db
spring.datasource.username=root
spring.datasource.password=your_password
```

### Run Application

```bash
mvn spring-boot:run
```

or run `EcommerceAppApplication.java` from Eclipse.
